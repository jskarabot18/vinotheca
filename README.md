# Vinotheca

**A library of wine tools — for the curious drinker.**

Live site: [jskarabot18.github.io/vinotheca](https://jskarabot18.github.io/vinotheca/)

A small collection of wine-exploration tools built by Jure Skarabot. Atlases, maps, a personal codex — each one approaches wine from a different direction.

## The four instruments

1. **[The Grand Cru Atlas](https://jskarabot18.github.io/grand-cru-atlas/)** — the world's greatest vineyards, arranged by grape variety.
2. **[TasteRank Explorer](https://jskarabot18.github.io/tasterank-explorer/)** — a similarity network of 101 grape varieties.
3. **[The Soul of Wine](https://jskarabot18.github.io/soul-of-wine/)** — an anthropological study of 59 wine regions and their identities.
4. **[Codex Vini](https://jskarabot18.github.io/codex-vini/)** — a personal wine atlas of bottles tasted.

## About this repository

This repository hosts the landing page that collects and introduces the four projects. Each of the tools above lives in its own repository.

The page is a single self-contained `index.html` file published via GitHub Pages. No build step, no dependencies.
